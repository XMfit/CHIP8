# CHIP8
An emulator for the CHIP-8 interpreter for educational purposes to further
my understanding of C and emulators.

## Requirements
- SDL Library
- C

Compiled on `Ubuntu 24.04` with `gcc version 13.3.0`

## Buildings 
`make all`

`make clean`

## Usage 
`./bin/emulator <filepath to rom>`
